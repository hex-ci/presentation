Web Audio 是什么


Web Audio 是一套在 Web 上处理音频的 API，使用这套 API 可以实现很多功能

比如：音乐可视化、音色合成器、动态混音、声音特效、3D空间音频、均衡器、环境混响等

可以应用在音乐播放器、电子音乐软件、游戏音效、音乐游戏、直播互动等领域

音频上下文 AudioContext


AudioContext 为音频处理提供一个上下文环境
相当于一个中央控制器
控制着音频路由图中的各个音频模块


处理音频都需要创建一个 AudioContext 实例，可全局共享
音频节点需要包含在 AudioContext 中
每个音频节点只能关联一个 AudioContext
